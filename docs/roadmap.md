# Learning Roadmap

This roadmap captures priorities for professional learning and open source contribution for 2026.

## Short-term (Next 6 months)
- Deepen knowledge of Angular Signals and performance patterns.
- Launch a public component scaffolding CLI for enterprise teams.
- Publish a technical write-up on migration strategies and codemods.

## Mid-term (6–18 months)
- Build a set of reusable, tested enterprise components with strong accessibility guarantees.
- Create a tested sample application that showcases modernization best practices.

## Open Source Goals
- Share migration tooling and testing harnesses for the community.
- Foster a small contributor-friendly repo with clear contribution guidelines.
